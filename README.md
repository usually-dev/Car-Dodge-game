# 🚗 Car Dodge Game

A simple yet fun **2D car dodging game** built using **HTML, CSS, and JavaScript (Canvas API)**.

---

## 🎮 Features

* 🛣️ 4-lane road system
* 🚗 Smooth player movement (lane-based)
* 🤖 Smart enemy cars (random + player-lane bias)
* 🎵 Background music + sound effects
* ❤️ Lives system (heart icons)
* ▶️⏸️ Play / Pause functionality
* 📱 Responsive (works on mobile + desktop)
* 🛣️ Animated road (moving dashed lines)
* 🎯 Increasing difficulty over time

## 🎯 How to Play

* ⬅️ Move left → Arrow Left
* ➡️ Move right → Arrow Right
* 🖱️ Click → Start / Restart
* ▶️⏸️ Click top-right icon → Pause / Resume

Avoid enemy cars and survive as long as possible!
